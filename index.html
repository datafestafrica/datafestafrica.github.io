Research, Data Analysis & Visualization

Data flows through the wold everyday, however what makes it valuable is the insights it generates.

For Business: This track explores the strategies, tools, skills and talent required to generate insights for optimal decision making for businesses.

For Research: It also explores the strategies, tools and skills required.

  • Visualisation
  • Data modelling
  • Metrics
  • Analytics
  • Research and analysis
  • Descriptive and inferential statistics
  • Location intelligence / Spatial analysis

ML/AI

For Business: Developing and operationalizing data science and machine learning (ML) is a huge task. This track is dedicated to exploring the strategies, tools, skills and talent needed for successful projects.

For Research: It is also dedicated to exploring strategies, tools and skills required for making ground breaking research in the ML and AI space

  • Models
  • Feature engineering
  • Model deployment
  • MLOps
  • NLP
  • Computer vision
  • Research and analysis

Data Management

Data has become the most valuable asset of an organization and the need for the optimal management of this asset cannot be overemphasized. This spotlight track highlights all the specific content about ranging from data quality, data strategy to data governance.

  • Data quality
  • Data strategy
  • Meta data management
  • Data governance
  • Data Lineage
  • Data catalog

Data Engineering, Analytics engineering and Infrastructure

Many organisations are making the transition into being data-driven. This track explores strategies, tools, skills and talent required to build a world class infrastructure. It also explores how data flows within an organisation from ingestion (raw data) to consumption (valuable insights).

  • Pipelines
  • Orchestration
  • Modern data stack
  • Analytics Engineering
  • Cloud/On-premise (AWS, Azure and GCP)
  • Startup VS Enterprise
  • DBT

Low Code Applications

Real world data problems require a combination of powerful tools. This track explores Power platform which helps organisations easily manipulate, automate and analyse data.

  • Data Solutions
  • Data Automation
  • Power Apps
  • Power Automate
  • Low Code Tools

Data Culture and Teams

Data proficeincy is no longer a capability limited to data teams. Tool stacks are becoming more friendly to everyday users and organisations are starting to see the importance of driving data literacy across the entire enterprise. This track explores themes such as data literacy, managing data teams, data culture and democratisation.

  • Data Literacy
  • Data Discovery
  • Data Democratisation
  • Data Culture
  • Data Teams
  • Citizen Data professionals
  • Self service

Blockchain Analytics

In recent times, we've seen wide applications of the Blockchain technology. There are number of decentralized applications been built, and deployed. The technique of analyzing, recognizing, and 'clustering' data on the blockchain - which is a cryptographic distributed-ledger available to everybody - is known as blockchain analytics. How can we leverage data in order to find crucial information about users and transactions? This aims to explore interesting thoughts on Blockchain analytics.

  • Blockchain computational power
  • Blockchain cost
  • Blockchain analytic platforms
  • Blockchain analytic tools

Data Communities and Data Careers

As global demand for data professionals grows, the supply also needs to match this demand. Data Communities are becoming more important in helping to bridge this gap. This track explores strategies and tips required to build world class communities and develop world class talent to match the growing talent demand.

  • Data communities
  • Data events
  • Data Projects
  • Data Portfolio
  • Data CV
  • Collaboration
  • Opportunities
